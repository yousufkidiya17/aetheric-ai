# 🌟 Aetheric — AI Service Marketplace

AI-powered service marketplace. Order food, book rides, hire workers — all through voice and chat.

## Deploy to Render.com (One Click)

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy)

## Features
- 🍕 Order Food (Mock Zomato/Swiggy)
- 🚕 Book Rides (Mock Ola/Uber)  
- 🔧 Hire Workers (Electrician, Plumber, Tutor)
- 🎤 Voice Input (Web Speech API)
- ⚡ Real-time WebSocket notifications
- 🤖 Mistral AI powered chat

## Environment Variables
Set `MISTRAL_API_KEY` in your hosting platform's environment settings.
